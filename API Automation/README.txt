Automated Test for Best Buy APIs:

In this folder you will find Three files: BestBuy.postman_collection, CreateStore.json & Testcases.docx

Before starting running the tests plese follow this link to start the server:
https://github.com/bestbuy/api-playground/
----------------------------------------------------------------------------------------------------
Step one: 
Open Postman and import the file BestBuy.postman_collection

Step Two:
Open Postman Runner from File>New Runner Tab

Step Three:
Choose the first folder inside the collection named: Products API and Start run

Step Four:
Open a new runner tab like step two and choose the second folder named: Stores API

Step Five: 
Select file and choose the file named: CreateStore.json and Start Run
